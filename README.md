#  C++ Celestial Motion Simulator
> **C++ 객체지향 프로그래밍을 활용한 천체 운동 및 달의 위상 변화 시뮬레이션**

##  개요
- **성격:** C++ 프로그래밍 교과 과정 2인 팀 프로젝트 및 개별 과제
- **언어:** C++
- **주요 목표:** 천체의 운동을 콘솔 환경에서 시각화

##  주요 수행 내용

### 1. C++ 핵심 메커니즘 숙달 (개별 과제)
- **메모리 관리:** 동적 할당(Dynamic Allocation)과 포인터(Pointer)를 활용한 효율적인 메모리 제어 로직 구현.
- **객체지향 설계:** 클래스(Class)의 상속과 캡슐화를 통해 유지보수가 용이한 코드 구조 설계 실습.

### 2. 천체 운동 시뮬레이션 (2인 팀 프로젝트)
- **콘솔 시각화 시스템 기획:** 텍스트 기반의 제한된 콘솔 환경에서 달의 위상 변화와 행성의 궤도 운동을 효과적으로 표현하기 위한 **출력 방식 및 UI 레이아웃 구상**.
- **배경 지식 조사 및 데이터화:** 천문학적 근거(케플러 법칙, 위상 변화 원리)를 조사하여 시뮬레이션의 물리적 타당성 확보.
- **핵심 로직 구현:** 시간의 흐름에 따른 달의 위상를 콘솔 화면에 시각화.
- **성과 공유:** 프로젝트 기획 의도와 기술적 구현 과정을 담은 PPT 제작 및 발표.

##  성과
- **추상화 및 구조화 역량:** 복잡한 자연 현상을 클래스와 객체 단위로 추상화하여 시스템화하는 경험을 통해, 게임 내 복잡한 콘텐츠를 구조적으로 설계하는 기틀을 마련함.
- **UX/UI 문제 해결:** 그래픽 라이브러리 없이 텍스트만으로 정보를 전달해야 하는 환경에서 최적의 시각적 전달 방식을 고민하며 사용자 경험(UX) 중심의 사고 역량을 강화함.
